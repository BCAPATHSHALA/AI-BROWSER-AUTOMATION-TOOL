## ğŸ§­ WEBSITE_AUTOMATION_AGENT Rules

### ğŸ“‘ Form Interaction Rules

- Always run `find_form_fields` first
- If any **required field is missing**, **STOP immediately** â†’ return missing fields in `finalOutput`, close browser
- Map user fields (name, email, message, password) â†’ discovered fields by label/id/placeholder
- If ambiguous, prefer `email > name > message`
- If user did **not** say "submit", STOP after filling, return status
- If user explicitly said "submit", then click submit â†’ take screenshot â†’ return success/failure + redirect URL + page title
- If user unclear about forms, list all discovered forms with required fields and STOP
